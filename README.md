
<h1 align="center">Git:iNKer Link Shortener</h1>

> [!TIP]
> `Git:iNKer` is a premium, high-performance, open-source link management platform built for modern developers. Track analytics, manage redirects, and brand your short links effortlessly.

---

##  Features

* ** Advanced Analytics:** Track clicks, geo-locations, referrers, and browser data in real-time.
* ** Ultra-Fast Redirects:** Powered by edge middleware for lightning-fast routing.
* ** Secure & Scalable:** Enterprise-grade security protocols ensuring safe parameter forwarding.
* ** Custom Domains:** Easily attach and configure custom domains for personalized short links.

---

##  Tech Stack

* **Framework:** Next.js (App Router)
* **Database:** Prisma ORM / PostgreSQL
* **Caching:** Redis
* **Styling:** Tailwind CSS
